History-enabled Scrollbar
Return-navigation study: standard vs history-enabled scrollbar (A/B)

User study (Milestone 2)

This study tests returning to places you already scrolled to. Each block has two guided visits (your exact position is saved), then two tasks where you must go back to those saved spots. Condition A is a normal scrollbar only; Condition B adds the history overlay (markers you can click). Order is counterbalanced (A→B or B→A). After each block, three short ratings. Then download the JSON for analysis.

Jumping back to spots I had already scrolled to felt easy with this scrollbar.
This scrollbar felt necessary to complete the return tasks without excessive trial and error.
